## Overview
Focused on building a digital checklist platform for ramp operation inspections, supporting daily employee submissions and centralized administrator management.

## Tech Stack
Node.js, Express, MongoDB, HTML, CSS, JavaScript, JWT, Multer.

## Responsibilities
- Responsible for overall frontend and backend setup.
- Built backend services and APIs with Express and used MongoDB for business data storage and management.
- Developed employee-side and admin-side interfaces with native HTML, CSS, and JavaScript.
- Implemented authentication, dynamic checklist rendering, record submission, photo upload, template configuration, statistical analysis, AI-generated reports, and issue closed-loop tracking.
