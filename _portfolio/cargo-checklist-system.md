## Overview
Built a digital checklist platform for daily inspection reporting and administrator-side unified management, integrating checklist submission, template management, record querying, issue tracking, and analytics.

## Tech Stack
Node.js, Express, MongoDB, HTML, CSS, JavaScript, JWT, Multer.

## My Role
- Built backend services and API flows with Express and MongoDB.
- Implemented employee-side and admin-side interactions with native HTML, CSS, and JavaScript.
- Supported authentication, dynamic checklist rendering, record submission, photo upload, template configuration, AI-assisted reporting, and issue closed-loop tracking.
