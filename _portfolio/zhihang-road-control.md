## Overview
Combined BEV hybrid algorithms, visual recognition, and data analysis to enable efficient recognition and real-time monitoring of multiple vehicle types.

## Tech Stack
PyQt5, PyAV, OpenCV, NumPy, pickle.

## Responsibilities
- Responsible for overall desktop UI interaction and core feature implementation.
- Built the desktop-side interface with PyQt5.
- Used PyAV and OpenCV for local video parsing, real-time camera capture, frame refresh, and playback control.
- Implemented video recording, file storage, and historical playback.
