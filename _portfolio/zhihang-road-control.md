## Overview
Built an intelligent vehicle detection desktop system for large transportation hubs, combining multi-type vehicle recognition, local video analysis, and real-time front-end interaction.

## Tech Stack
PyQt5, PyAV, OpenCV, NumPy, pickle.

## My Role
- Developed modules for login/registration, user management, and desktop-side human-computer interaction.
- Implemented local video parsing, live camera capture, frame refresh, playback control, and progress dragging.
- Supported video recording, file storage, and historical playback to improve system stability and usability.
