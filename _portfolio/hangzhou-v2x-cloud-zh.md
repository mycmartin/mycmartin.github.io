## 项目概述
参与“车（空）-路（地）-云-网-图”体系建设，在杭州打造车路协同和空地协同一体化示范区，支持智能网联测试、低空运输与城市智慧治理。

## 技术栈
Vue 3、JavaScript、CSS、Ant Design Vue、Vue Router、MQTT。

## 个人主要工作
- 基于 Vue 3 搭建整体页面架构，完成登录、个人中心、预警提醒、数据展示等模块开发。
- 使用 Vue Router 实现路由管理与业务跳转。
- 基于 MQTT 实现实时消息订阅、状态更新与告警推送，并配合多方接口联调完成数据可视化呈现。
