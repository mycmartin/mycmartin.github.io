## 项目概述
建设“车（空）-路（地）-云-网-图”体系，支持智能网联测试、低空运输与城市智慧治理。

## 技术栈
Vue 3、JavaScript、CSS。

## 个人主要工作
- 我主要负责平台前端页面与核心交互能力的实现。
- 基于 Vue 3 搭建整体页面架构，完成登录、个人中心、预警提醒、数据展示等模块开发。
- 同时与百度，阿里，智能网联，华为等公司和政府部门的进行接口数据对接联调。
- 完成监测数据、风险预警及可视化信息的前端呈现。
