## Overview
Fine-tuned a Qwen-based multimodal large model for agricultural scenarios, supporting both image and text inputs.

## Tech Stack
Python, LLaMA-Factory.

## Responsibilities
- Built the dataset through on-site greenhouse photography.
- Constructed the knowledge base.
- Tuned hyperparameters.
- Deployed the model on an edge processor based on Jetson AGX Orin.
